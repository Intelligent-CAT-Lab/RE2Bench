1. The function `get_fontconfig_pattern` is called from `self` (a FontProperties instance).  
   - Inside `generate_fontconfig_pattern`, `d` is `self`.  
   - `kvs` is built as:  
     For each `k` in ["style", "variant", "weight", "stretch", "file", "size"],  
     get `getattr(self, f"get_{k}")()`.  

   From the given `self` attributes:  
   - `self._family = ["Noto Sans CJK SC"]` → `get_family()` returns this list.  
   - `self._slant = "normal"` → `get_style()` returns `"normal"`.  
   - `self._variant = "normal"` → `get_variant()` returns `"normal"`.  
   - `self._weight = "normal"` → `get_weight()` returns `"normal"`.  
   - `self._stretch = "normal"` → `get_stretch()` returns `"normal"`.  
   - `self._file = null` → `get_file()` returns `None`.  
   - `self._size = 12.0` → `get_size()` returns `12.0`.  

2. Build `kvs`:  
   - ("style", "normal")  
   - ("variant", "normal")  
   - ("weight", "normal")  
   - ("stretch", "normal")  
   - ("file", None) → will be skipped because `v is not None` is False.  
   - ("size", 12.0)  

3. Generate the pattern string:  
   - Family part: `",".join(_family_escape(f) for f in d.get_family())`  
     → `get_family()` returns `["Noto Sans CJK SC"]` → `_family_escape("Noto Sans CJK SC")`.  
     Since `_family_escape` likely just returns the string unchanged (or wraps with quotes if needed),  
     we get `"Noto Sans CJK SC"`.  

   - For other non-None entries in `kvs`:  
     `("style", "normal")` → `":style=normal"`  
     `("variant", "normal")` → `":variant=normal"`  
     `("weight", "normal")` → `":weight=normal"`  
     `("stretch", "normal")` → `":stretch=normal"`  
     `("size", 12.0)` → `":size=12.0"` (since `str(12.0) = "12.0"`).  

   - Concatenate family part and the rest without space:  
     `"Noto Sans CJK SC:style=normal:variant=normal:weight=normal:stretch=normal:size=12.0"`.  

4. So `get_fontconfig_pattern()` returns that string.
